El bloque básico de un documento es el párrafo. Dentro de un documento HTML, es preciso marcar explícitamente el comienzo de un párrafo, para lo cual se utiliza la etiqueta En caso de no especificarlo, todo el texto se mostraría como una única línea. Se debe tener en cuenta que determinados elementos que se verán más adelante (títulos, blockquotes, preformateados, listas) insertan un salto de párrafo o línea de forma automática.
