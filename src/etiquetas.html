Uso de Etiquetas de Formato

Voy a hablar sobre las etiquetas destinadas a dar sentido a una parte concreta de la página web:

Ejemplos de etiquetas:

"em" se emplea para enfatizar un texto; habitualmente se representa en cursiva: "ejemplo"

"i" se utiliza para darle importancia al texto: "ejemplo"

"dfn" se utiliza para definiciones: "ejemplo"

"code" se utiliza para bloques de código informático: #include main() { for(;;) { printf ("Hello World!\n"); } }

"samp" se utiliza para ejemplos: "Presione f1"

"kbd" se utiliza para texto de teclado, ejemplo: "Si está conforme escriba sí. Si no, pulse esc"

"var" se utiliza para hipertextos: "Ejemplo"

"cite" Se utiliza para una cita en hipertexto, ejemplo: Galileo dijo: "... y sin embargo, se mueve."

"pre" representa texto preformateado, ejemplo: 

            body{
              color:  red;
            }
            a   {
              color:green;
            }

"span"Sirve para aplicar estilo al texto o agrupar elementos en línea, ejemplo: y esto un span dentro de un párrafo.

"mark" se utiliza para marcar un texto importante: "Ejemplo"

"small" se utiliza para representar mas pequeño un texto: "Ejemplo"

"del" se utiliza para tachar un texto como eliminado: "Ejemplo"

"ins" se utiliza para subrayar texto: "Ejemplo"

"sub" se utiliza para subindices, ejemplo: La fórmula química de agua: H2O

"sup" se utiliza para superindices, ejemplo: Notas al pie1

"b" se utiliza para poner la letra en negrita, aunque su uso es desanconsejable: Ejemplo
